TEF B1 高频互动刷题系统 v1.0

使用方式：
1. 打开 index.html
2. 演示账号：demo
3. 演示密码：tef2026
4. 登录后进入 app.html

已做功能：
- 登录页
- 账号有效期检查
- 当前浏览器设备绑定
- 动态身份水印
- 阅读 / 语法 / 听力入口
- 本地错题板
- 禁止右键 / 禁止复制 / 禁止打印基础防护
- 切屏或离开窗口时页面模糊

重要限制：
- 这是前端本地原型，不是真正可售卖的后端账号系统。
- 一账号一设备在本地版只对同一浏览器有效，不能阻止另一台电脑重新绑定。
- 正式售卖必须接 Supabase / 后端数据库，把账号、到期、设备绑定和题库下发放到服务器。

正式版建议：
- Supabase Auth：账号登录
- user_access 表：到期控制
- user_devices 表：设备绑定
- user_progress 表：云端错题和进度
- Edge Function：每隔几分钟进行服务端权限校验
